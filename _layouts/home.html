The Möbius Project

Computing for all.

What is Mobius?

Mobius is a customizable visual programming environment suitable for all levels of programming expertise. With the need for coding fast performating every domainm we felt it was unrealistic to expect every person to get down to the nitty gritties of code to harness this power. Mobius promotes computation thinking with reusable modules and a drill-down approach, allowing users at various levels of expertise to interact with the code at a level germane to their usage - the output, the workflow or right down to the controls.

Mobius Components

looks_onePresentation Layer

Visualize your content in your own way, with custom viewers or default viewers that come with Mobius. This upper layer, hides all processing underneath and allows you to share your final result with the world.

looks_twoFlow-based Editor

For the domain specialists. Re-use code and use precreated nodes to create variations of the same workflow. Completely customize your own node library suited to the needs of your business. If you work out your process once, there is no reason why you shouldn't be able to go from raw input to the desired output in a matter of minutes.

looks_3Code Editor

Hidden away in the final depths, yet present to allow advanced users absolute control over the environment.
